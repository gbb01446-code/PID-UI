# PID_UI Widget

EdgeTX 2.12 / Rotorflight 2.2 対応のパラメータ調整・表示Widgetです。

送信機のスイッチ操作だけで、フライト中にRotorflightのPIDやレート、送信機側パラメータをリアルタイムに調整・表示・保存できます。

---

## 主な機能

**RF2 Adjustmentsモード（SD手前）**
- 3×3のスイッチグリッドで9種類のパラメータを選択
- トリムスイッチで値をステップ増減
- AdjVテレメトリーで確定値を受信し、自動保存
- Ail / Ele / Rud の3軸を個別に表示・管理

**TX Setモード（SD中・SD奥）**
- スロットル回転数（RPM表示）
- コレクティブピッチ（度数表示）
- サイクリックExpo / ラダーExpo
- フライトモード（FM0〜2）別に値を管理

**共通機能**
- バッテリー電圧・残量・RPM・タイマーの表示
- フライトモード別バンク管理（FM0〜2）
- パラメータ変更時の音声読み上げ
- 設定値のファイル自動保存

```
    │   ├── step6.wav     # I-Gain読み上げ音

    │   ├── step8.wav     # FeedForward読み上げ音
    │   ├── step9.wav     # B-Gain読み上げ音
    │   ├── step10.wav    # Stop-Gain読み上げ音
    │   └── step11.wav    # (未使用 / 調整無効)

```

## スイッチ構成（概要）

| スイッチ | 役割 |
|---------|------|
| SD奥 | TX Set表示（参照のみ） |
| SD中 | TX Setモード（パラメータ調整可） |
| SD手前 | RF2 Adjustmentsモード |
| SB × SA | 3×3グリッドでパラメータ選択 |
| Ailトリム | Ail軸 / サイクリックExpo / 回転数調整 |
| Eleトリム | Ele軸 / コレクティブ調整 |
| Rudトリム | Rud軸 / ラダーExpo調整 |

詳細は `SETUP.md` および `REFERENCE.md` を参照してください。
